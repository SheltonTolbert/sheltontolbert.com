Emission

The whole lighting process starts with an emission source. In the real world this can be a light bulb, the sun or anything else that produces light (this article covers real world emissive sources https://bit.ly/2N9HggB). Just as in the real world, in computer graphics an emissive source is required for an image to render properly. These emissive sources can be either light sources or self-emissive materials. Computer graphics light sources function much like their physical counterparts, a spot light will function as expected, as will a point or directional light (a directional light is a light source that is infinitely far away - think the sun). There is one light source that exists in computer graphics that is technically physically impossibly - the ambient light source. The ambient light source blankets the entire virtual world with the same amount of light, this makes shadows impossible but serves as a method of providing a base light level (floor) that the developer can then add other light sources to. 

Emission & Self-Emission

Self emission can be described as any material that can emit light directly, the tungsten in a light bulb for example. While most materials in the real world are not emissive, in computer graphics we can add an emissive value to any material we want. This is particularly useful when creating motion graphics or any other non-photorealistic images.  

 

Absorption & Reflection

Light absorption is the process by which certain frequencies of the visible light spectrum are absorbed by the surface of an object. This light absorption is how color is calculated as any frequencies that are not absorbed are reflected, these reflected frequencies determine the color of the object. When we set the color value of a material we are actually dictating which frequencies will be reflected( color(255,0,0) will reflect only red values). When we set the color of a light we dictate the frequencies of emission (light.color(0,0,255) will emit blue).  Note that if the light is emitting red and theobject material only reflects blue, the object will not reflect any light and as such will appear black.

An Interactive Demonstration

The following p5 example illustrates the different types of light sources and materials and their interactions. The top three sliders adjust the x,y,z position of the point light source, the buttons will allow you to change the light source/material/enable object wireframe, and the bottom slider controls the global light value. Note that the ambient light and emissive material are uniform light sources therefore all object outline will be lost. By enabling wireframe mode you can see the  details of the environment in these conditions. The camera can be controlled via click and drag orbit.
